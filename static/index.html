OS/ブラウザが鳴らしてる音声を getDisplayMedia で取得し, 5 秒チャンクで Gemini CLI に送信して音楽解析させる実験. 解析結果 (キー / BPM / 楽器 / 旋律 / 雰囲気) がリアルタイム風にフィードされる.

idle
🎧 使い方: START → 「タブを共有」か「画面全体を共有」+ 「音声も共有」チェック必須 → 音楽を鳴らす.
⚠ Chrome / Edge のみ. Firefox は system audio 非対応.
⚠ 最初の解析は 10-20 秒遅れ (チャンク長 5s + Gemini 推論時間).
