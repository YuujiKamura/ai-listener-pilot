PILOT · GEMINI EAR (record → stop → analyze)

OS/ブラウザが鳴らしてる音声を getDisplayMedia で取得し, STOP を押した時点で全部まとめて Gemini CLI に食わせる. Gemini の stdout は下のターミナル風パネルにストリーミング表示される.

00:00
idle
gemini --yolo -p "..."
🎧 使い方: REC → 共有ダイアログ「タブを共有」or「画面全体」+ 「音声も共有」チェック必須 → 音楽再生 → STOP & ANALYZE で Gemini 解析.
⚠ Chrome / Edge のみ. Firefox は system audio 非対応.
⚠ Gemini が応答しない場合は server コンソールを確認 (PATH 解決 / OAuth 状況).
